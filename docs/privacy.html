About this app

Spoolman GO is an iPhone companion app for managing 3D printing filament spools. It connects to a self-hosted Spoolman-compatible or FilaMan-compatible server that you operate yourself. The developer does not operate a backend service for normal app use.

Data collected by the developer

The developer of Spoolman GO does not collect, transmit, or store any personal data on external servers. No analytics SDKs, crash-reporting services, advertising frameworks, or third-party tracking are included in the app.

Data stored locally on your device

The following data is saved exclusively in the app's local storage on your device:

DataPurpose
Server URLConnecting to your self-hosted Spoolman/FilaMan instance
API profileDetected API type (Spoolman or FilaMan)
Theme preferenceLight/dark/system appearance
Language preferenceApp language override (system, English, German)
Low-stock thresholdWarning threshold for nearly empty spools
Onboarding stateWhether first-launch setup has been completed
App modeDemo Mode or Live Mode

This data never leaves your device unless you explicitly point the app to an external server.

Communication with your server

In Live Mode, the app connects directly to the Spoolman-compatible server URL you provide. All filament, spool, vendor, and location data is fetched from and written to that server. The app developer has no access to this data.

In Demo Mode, the app operates entirely offline using built-in sample data. No network requests are made.

NFC

The app can read and write NFC tags for spool identification using the OpenSpool format. NFC tag data is only read from or written to physical tags in your possession. No NFC data is transmitted externally.

Security recommendations

  • Use the app on a local network or VPN rather than a publicly accessible Spoolman instance.
  • Prefer HTTPS connections for any public-facing server setup.
  • The app warns you when an unencrypted HTTP connection is configured.

Children's privacy

The app is not directed at children under 13 and does not knowingly collect data from children.

Changes to this policy

Any future changes will be noted with an updated version date at the top of this document and reflected in the App Store listing.

Über diese App

Spoolman GO ist eine iPhone-Begleit-App zur Verwaltung von 3D-Druck-Filamentspulen. Die App verbindet sich mit einem selbst gehosteten Spoolman-kompatiblen oder FilaMan-kompatiblen Server, den du selbst betreibst. Der Entwickler betreibt keinen eigenen Backend-Dienst.

Vom Entwickler erhobene Daten

Der Entwickler von Spoolman GO erhebt, überträgt oder speichert keine personenbezogenen Daten auf externen Servern. Die App enthält keine Analyse-SDKs, Absturzmeldedienste, Werbeframeworks oder Drittanbieter-Tracking.

Lokal auf deinem Gerät gespeicherte Daten

Die folgenden Daten werden ausschließlich im lokalen App-Speicher auf deinem Gerät gespeichert:

DatenZweck
Server-URLVerbindung mit deiner selbst gehosteten Spoolman/FilaMan-Instanz
API-ProfilErkannter API-Typ (Spoolman oder FilaMan)
Design-EinstellungHell/Dunkel/System-Darstellung
SpracheinstellungApp-Sprache (System, Englisch, Deutsch)
Fast-leer-WarnschwelleWarnschwelle für fast leere Spulen
Onboarding-StatusOb die Ersteinrichtung abgeschlossen wurde
App-ModusDemo-Modus oder Live-Modus

Diese Daten verlassen dein Gerät nicht, außer du verbindest die App ausdrücklich mit einem externen Server.

Kommunikation mit deinem Server

Im Live-Modus verbindet sich die App direkt mit der von dir angegebenen Spoolman-kompatiblen Server-URL. Alle Filament-, Spulen-, Hersteller- und Standortdaten werden von diesem Server abgerufen und dorthin geschrieben. Der App-Entwickler hat keinen Zugriff auf diese Daten.

Im Demo-Modus arbeitet die App vollständig offline mit integrierten Beispieldaten. Es werden keine Netzwerkanfragen gestellt.

NFC

Die App kann NFC-Tags zur Spulenidentifikation im OpenSpool-Format lesen und beschreiben. NFC-Tag-Daten werden ausschließlich von physischen Tags in deinem Besitz gelesen oder auf diese geschrieben. Es werden keine NFC-Daten extern übertragen.

Sicherheitshinweise

  • Verwende die App im lokalen Netzwerk oder über VPN, nicht über eine öffentlich zugängliche Spoolman-Instanz.
  • Bevorzuge HTTPS-Verbindungen bei öffentlich erreichbaren Serversetups.
  • Die App warnt dich, wenn eine unverschlüsselte HTTP-Verbindung konfiguriert ist.

Datenschutz für Kinder

Die App richtet sich nicht an Kinder unter 13 Jahren und erhebt wissentlich keine Daten von Kindern.
